设计模式-创建型-工厂方法(Factory Mehtod)

发表于 2016-07-10   |   分类于 技术   |  

使用工厂方法原因

  在Java中，新建对象是通过修饰符new + "className"来实现的。但是这样存在一个问题，使用new实例化一个具体类是使用了 实现，而非接口。当加入新的具体类的时候，就必须修改代码，违反了 开闭原则。我们将新建具体类的代码封装到一个类里面，这个类就是工厂(fatory)。

工厂方法的分类

  工厂方法模式分为三种：简单工厂模式、工厂方法模式、静态工厂模式，其中用的最好的是静态工厂模式

简单工厂模式

概述

  定义一个工厂类，它可以根据参数的不同返回不同类的实例，被创建的实例通常都具有共同的父类。

UML图

典型代码

典型的抽象产品类代码如下所示：

public abstract class AbstractProduct {
}

典型的具体产品类代码如下所示：

public class ConcreteProductA extends AbstractProduct {
}

public class ConcreteProductB extends AbstractProduct {
}

典型的工厂类代码如下所示：

public class Factory {
    public AbstractProduct produce(String type) {
        AbstractProduct product = null;
        if ("1".equals(type)) {
            product = new ConcreteProductA();
        } else if ("2".equals(type)) {
            product = new ConcreteProductB();
        }
        return product;
    }
}

典型的客户端类代码如下所示：

public class App {
    public static void main(String[] args) {
        AbstractProduct product;
        Factory factory = new Factory();
        product = factory.produce("A");
        System.out.println(product instanceof ConcreteProductA);
    }
}

示例代码

抽象产品类-MobilePhone

public abstract class MobilePhone {
    protected OS os;
    public OS getOs() {
        return os;
    }
    public void setOs(OS os) {
        this.os = os;
    }
}

具体产品类-AndroidPhone

public class AndroidPhone extends MobilePhone {
    public AndroidPhone() {
        this.os = OS.Android;
    }
}

具体产品类-IOSPhone

public class IOSPhone extends MobilePhone {
    public IOSPhone() {
        this.os = OS.IOS;
    }
}

工厂类-MobilePhoneFactory

public class MobilePhoneFactory {
    public MobilePhone produce(String os) {
        MobilePhone mobilePhone = null;
        if ("android".equalsIgnoreCase(os)) {
            mobilePhone = new AndroidPhone();
        } else if ("ios".equalsIgnoreCase(os)) {
            mobilePhone = new IOSPhone();
        }
        return mobilePhone;
    }
}

工厂方法模式:相对简单工厂模式只是工厂类发生了变化

UML图

典型代码

典型的工厂类代码如下所示：

public class Factory {
    public AbstractProduct produceProductA() {
        return new ConcreteProductA();
    }

    public AbstractProduct produceProductB() {
        return new ConcreteProductB();
    }
}

典型的客户端类代码如下所示：

public class App {
    public static void main(String[] args) {
        AbstractProduct product;
        Factory factory = new Factory();
        product = factory.produceProductA();
        AbstractProduct product2 = factory.produceProductB();
    }
}

示例UML图

示例代码

抽象产品类-Cola

public abstract class Cola {
    private Size brand;
    public Cola(Size brand) {
        super();
        this.brand = brand;
    }
    @Override
    public String toString() {
        return this.getClass().getSimpleName() + " [brand=" + brand + "]";
    }
}

具体产品类-CocaCola和PensiCola

public class CocaCola extends Cola {
    public CocaCola(Size os) {
        super(os);
    }
}

public class PensiCola extends Cola {
    public PensiCola(Size brand) {
        super(brand);
    }
}

工厂类-ColaFactory

public class ColaFactory {
    public Cola producePensiCola(Size size) {
        return new CocaCola(size);
    }
    public Cola produceCocaCola(Size size) {
        return new PensiCola(size);
    }
}

客户端类-MultiFactoryApp

public class MultiFactoryApp {
    public static void main(String[] args) {
        ColaFactory factory = new ColaFactory();
        Cola firstCola = factory.produceCocaCola(Size.BOTTLING);
        System.out.println(firstCola);
        Cola secondCola = factory.producePensiCola(Size.CANNED);
        System.out.println(secondCola);
    }
}

public abstract class Human {
    private Color color;
    public Human(Color gender) {
        super();
        this.color = gender;
    }
    @Override
    public String toString() {
        return this.getClass().getSimpleName() + " [gender=" + color + "]";
    }
}

具体产品类-CocaCola和PensiCola

public class Man extends Human {
    public Man(Color gender) {
        super(gender);
    }
}

public class Woman extends Human {
    public Woman(Color gender) {
        super(gender);
    }
}

工厂类-ColaFactory

public class GodFactory {
    public static Human produceMan(Color color) {
        return new Man(color);
    }
    public static Human produceWoman(Color color) {
        return new Woman(color);
    }
}

客户端类-MultiFactoryApp

public class StaticApp {
    public static void main(String[] args) {
        Human produceMan = GodFactory.produceMan(Color.WHITE);
        System.out.println(produceMan);
        Human produceWoman = GodFactory.produceWoman(Color.BLACK);
        System.out.println(produceWoman);
    }
}
